## 简单说说测评结论

从第三方测评数据来看，MikuCloud 的几个特点比较一致：

- **电信、联通方向表现不错**，晚高峰跑大带宽问题不大；移动用户体验相对差一些，这点官方也没有回避过。
- **流媒体解锁能力强**，Netflix 港区、Disney+ 港区、TVBAnywhere+、ChatGPT 港区等，HGC 商宽和 FTTX 家宽产品表现尤其好。
- **磁盘 I/O 不错**，NVMe SSD 的随机读写对建站、跑服务都够用。
- **自持 ASN**，对 IP 洁净度有一定保障，不过具体每块 IP 的情况还是要自己测一下。

适合人群大概分两拨：一是需要香港节点、主要用电信/联通访问的个人和中小团队；二是有香港原生 IP 需求（流媒体解锁、跨境业务）的企业用户，后者可以直接上 FTTX 家宽或 IEPL 专线。
